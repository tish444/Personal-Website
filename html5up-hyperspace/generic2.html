Pool Game Simulator (2024)

For a school project, I developed a full-stack interactive pool game, employing a range of programming languages. The backend was implemented in C, the server was built using Python, and the frontend was crafted with HTML, CSS, and JavaScript. This project presented significant challenges but was highly rewarding, allowing me to enhance my portfolio with new programming languages and technologies.

In addition to development, we presented our fully functional game to a Teaching Assistant, who tested it and reviewed our work. We detailed our logic and code decisions, explaining the rationale behind our approach. This experience was invaluable for learning how to effectively articulate technical choices and furthered my growth in both programming and project presentation.
