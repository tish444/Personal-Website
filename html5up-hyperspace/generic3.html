Employer Assignment (2023)

This project was one of the first instances where I utilized file handling in a programming course. It involved using linked lists to read from and create information in text files. The program offered a variety of options to users; for example, to add an employee, users selected an option and provided the required details. To search for an employee, a specific choice was made, which then displayed information about that employee. The program also included functionality to "fire" a hypothetical employee, illustrating the dynamic nature of data management.

This project was both enjoyable and educational, significantly enhancing my understanding of linked lists and memory management in C. It provided practical experience in managing data and handling file operations, further solidifying my foundational knowledge in these areas.
